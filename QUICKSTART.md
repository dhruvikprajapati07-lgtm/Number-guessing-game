# ⚡ Quick Start Guide

## Installation & Running

```bash
# Simply run the game
python game.py
```

That's it! No installation or dependencies needed.

## Your First Game

1. When prompted, enter `1` for **Classic Mode**
2. Select `1` for **Easy** difficulty
3. Start guessing! You have 10 attempts
4. Type `hint` if you need help
5. Type `quit` to exit

## Menu Options Explained

```
1️⃣  Classic Mode      - Standard guessing, get feedback (HIGH/LOW)
2️⃣  Survival Mode     - Win 5 rounds in a row, increasing difficulty
3️⃣  Speed Mode        - Race against the clock
4️⃣  Multiplayer       - Compete with a friend
5️⃣  Reverse Mode      - You think of a number, AI guesses
6️⃣  Endless Mode      - Build a streak, gain points
7️⃣  Statistics        - View your game history & performance
```

## What to Expect

- **Feedback**: "📈 Too LOW!" or "📉 Too HIGH!"
- **Hints**: `💡 The number is even` (or other helpful info)
- **Scoring**: Based on difficulty and how quickly you guess
- **History**: Your stats are saved automatically

## Pro Tips

- 🎯 **Classic**: Save hints for when the range gets narrow
- 🔥 **Survival**: Play conservatively - one loss ends the game
- ⏱️  **Speed**: Make calculated guesses to manage time
- 👥 **Multiplayer**: Take turns strategically
- 🤖 **Reverse**: Give consistent hints to test the AI
- 🎪 **Endless**: Each new round gets harder!

## Statistics

Check your performance anytime from the main menu (option 7):
- Total wins/losses
- Best attempt record
- Performance by difficulty
- Favorite game mode

## Resetting Stats

Delete `game_stats.json` to clear all history and start fresh.
